Documentación de ycurve
========================

`ycurve` es una biblioteca de Python con el objetivo de implementar la aritmética
en cuerpos binarios y proveer la capacidad para definir curvas elípticas sobre ellos.
Además permite implementar algoritmos sobre ellas.
