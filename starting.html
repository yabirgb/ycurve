Comenzando con ycurve¶

ycurve es una librería escrita en python que permite trabajar con curvas elípticas en característica dos. El proyecto es de código abierto y se encuentra disponible en https://github.com/yabirgb/ycurve .

Para instalar la librería localmente podemos hacer:

git clone https://github.com/yabirgb/ycurve.git
cd ycurve
pip3 install -U .

O de manera directa:

pip install git+https://github.com/yabirgb/ycurve.git
